EDUCATION

I am currently studying information engineering at the Technische Universität München. This Bachelor focuses on the theoretical aspect of Computer Science with courses complementing the theoretical side through the application of the given theory (primarily through the use of JAVA). In addition to the theoretical and practical courses we also have several courses which focus on the business side of developing and implementing a digital system. This combination has enabled me to learn the full lifecycle of an information system, from the planning, development, implementation and then finally the maintenance involve with information system.

JAVA

During my time in TUM I have become intimately familiar with JAVA. From Cyber Physical Systems to Discrete Structures and Algorithms, JAVA has been the choosen tool to realise theory taught in lectures. Over this time I have pushed thousands of lines of code and learnt how powerful an Object Orientated Language can be when effectively implemented and been ever so grateful for the 8am hand-in times provided by our Professors.

FRONTEND DEVELOPMENT

Although I have not learnt HTML and CSS within TUM, I discovered the process of web-development through online resources. Frontend Development is the middle ground between the more technical coding which I have primarly learnt in school and a more artist side which programming can hold. This is what really drew me into frontend development the need for technical skills and understand but also an artistic eye which is need to have a good website. Although I feel I have only scratched the surface with front end development I am excited to excited to keep exploring and learning within this space.

PYTHON

Python was my first programming language, it was the introduction to the rest of my life in some ways. Although I now primarly use JAVA for most of my programming projects, I still find myself coming back to it when needing to make a simple program.
